Jeu de mots croisés

Le jeu implique le remplissage d'une grille où des mots s'entrelacent à la fois horizontalement et verticalement, en utilisant les définitions fournies pour chaque mot.

Lorsque vous cliquez sur une case contenant un numéro, toutes les cases du mot correspondant sont sélectionnées, permettant d'entrer le mot entier. En revanche, un simple clic sur les autres cases (les cases qui ne contiennent pas de numéro sauf les cases noires) sélectionne uniquement la case elle-même, pour y entrer une seule lettre.

Une fois la(les) case(s) sélectionnée(s), cliquez le bouton:

  • "Entrer un mot" pour entrer le mot ou la lettre
  • "modifier le mot" pour modifier le mot ou la lettre
  • "Effacer le mot" pour effacer le mot ou la lettre
  • "verifier les reponses" pour vérifier les mots entrés une fois que la grille est complète
  • "Réinitialiser la grille" pour effacer tous les mots de la grille et recommencer le jeu.

Une exception concerne le double-clic sur la case numéro 1 : cela sélectionne les cases du mot vertical 1.

  • 5- pronom interrogatif
